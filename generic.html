Détail du Projet
Plateforme de Gestion Immobilière

Application complète de gestion immobilière avec microservices

Description du Projet

Développement d'une plateforme complète de gestion immobilière permettant aux agences de gérer leurs biens, clients, rendez-vous et contrats. L'application comprend un backend en microservices Spring Boot, un frontend mobile Flutter et une interface web Angular.

Technologies Utilisées

Backend

  • Spring Boot (Microservices)
  • Spring Security avec JWT
  • PostgreSQL & MongoDB
  • Redis pour le cache
  • RabbitMQ pour la messagerie

Frontend & DevOps

  • Flutter pour mobile
  • Angular pour web
  • Docker & Docker Compose
  • GitHub Actions CI/CD
  • Nginx comme reverse proxy

Fonctionnalités Principales

  • Gestion complète des biens immobiliers
  • Système de rendez-vous en ligne
  • Gestion des clients et contrats
  • Notifications en temps réel
  • Tableaux de bord analytiques
  • Gestion des paiements

Architecture

L'application suit une architecture microservices avec 6 services indépendants :

  1. Service Authentification
  2. Service Gestion des Biens
  3. Service Gestion des Clients
  4. Service Rendez-vous
  5. Service Notification
  6. Service Reporting
