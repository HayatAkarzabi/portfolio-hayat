Description du Projet

Ce projet utilise des techniques de machine learning pour prédire les prix des appartements à Paris. Basé sur le dataset "Paris Housing" de Kaggle, j'ai développé plusieurs modèles de régression et créé une interface interactive avec Streamlit.

Dataset Utilisé

Source Nombre d'observations Features Période
Kaggle - Paris Housing 10,000+ 17 variables 2019-2020

Variables Principales

Caractéristiques du Bien

  • Surface (m²)
  • Nombre de pièces
  • Étage
  • Année de construction
  • Type de bien
  • Arrondissement

Caractéristiques du Quartier

  • Proximité transports
  • Nombre d'écoles
  • Commerces à proximité
  • Espaces verts
  • Indice de sécurité
  • Prix moyen du m²

Pipeline de Traitement

# Pipeline de prétraitement et modélisation
import pandas as pd
from sklearn.pipeline import Pipeline
from sklearn.compose import ColumnTransformer
from sklearn.preprocessing import StandardScaler, OneHotEncoder
from sklearn.ensemble import RandomForestRegressor
from sklearn.model_selection import train_test_split

# Préparation des données
numeric_features = ['squareMeters', 'numberOfRooms', 'floors']
categorical_features = ['cityCode', 'hasStormProtector']

# Pipeline de prétraitement
preprocessor = ColumnTransformer([
    ('num', StandardScaler(), numeric_features),
    ('cat', OneHotEncoder(), categorical_features)
])

# Pipeline complet
pipeline = Pipeline([
    ('preprocessor', preprocessor),
    ('regressor', RandomForestRegressor(n_estimators=100))
])

Interface Streamlit

J'ai développé une interface web interactive permettant aux utilisateurs de :

  • Saisir les caractéristiques d'un bien
  • Visualiser la prédiction en temps réel
  • Comparer différents arrondissements
  • Télécharger les prédictions en CSV
  • Visualiser l'importance des features

Comment lancer l'application :

# Installation
pip install -r requirements.txt

# Lancer l'application Streamlit
streamlit run app.py

# L'application sera disponible sur http://localhost:8501

Résultats et Insights

Insight 1 : Surface vs Prix

La surface est le facteur le plus important (42% de variance expliquée), suivie par l'arrondissement (28%).

Insight 2 : Effet de l'étage

Les étages élevés (sans être le dernier) ont une prime de +8% par rapport au rez-de-chaussée.

Insight 3 : Impact des transports

La proximité d'une station de métro ajoute en moyenne 12% à la valeur du bien.
